## 列表

### 有序列表

``` markdown
以数字N.开头
example
1. 打开冰箱
2. 把大象装进去
3. 关上冰箱
```

1. 打开冰箱

2. 把大象装进去
3. 关上冰箱

### 无序列表

``` markdown
以-开头或者*
example
- 打开冰箱
- 把大象装进去
- 关上冰箱
```

- 打开冰箱
- 把大象装进去
- 关上冰箱

### 任务列表

``` markdown
以-[] 开头
example
- [ ] 吃饭
- [ ] 睡觉
```

- [ ] 吃饭
- [ ] 睡觉

> 在中括号里填写x就会把勾打上
